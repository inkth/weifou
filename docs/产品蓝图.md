# 微否产品蓝图（v1 · 2026-06-29）

> 一页纸,讲清楚微否是什么、怎么增长、怎么赚钱。

## 一句话定位

**微否 = 会说话的你。**
一张能对话的 AI 名片:别人点开能直接「问你」;你越用它越懂你,也越多人看懂你。

- 是:数字世界里的你自己(身份),一张活的、可对话的个人名片 / 主页。
- 不是:客服工具、聊天机器人玩具、虚拟恋人。
- 北极星:**成为更好的、更被看见的自己。**

## 用户旅程

建分身(会说话的你)→ 镜子照见自己 → 把活名片发出去(社交 / 求职 / 相亲 / 获客)
→ 别人来问、来看 → 想被更多人看见 & 想变得更强 → **升级会员**。

入口不是「接管微信私聊」(平台封死),而是**社交本能**:任何要自我介绍的场合,甩一张「会说话的我」代替一段干巴简介。

## 免费 / 会员分层

| | 免费(增长引擎) | 会员(变现) |
|---|---|---|
| **身份** | 活名片、镜子、问答箱、被人问 | 发现墙曝光位、谁看过我、访客洞察、更强分身 |
| **成长** | 每个 AI 导师试用几次 | **AI 导师全家桶无限用**(英语 / 商业 / …)、吃画像的个性化教案 |

## 盈利飞轮

```
免费身份层(活名片·镜子·问答箱)
   → 病毒拉新 + 沉淀「你是谁」的画像数据
        → 会员变现层
            ① AI 导师全家桶(英语/商业/…)——吃分身画像做个性化
            ② 身份增值(曝光位 · 谁看过我 · 更强分身)
        → 更好的自己 → 更值得被展示 → 拉更多人进来
```

**为什么能赚、单做工具不能赚:**
- 获客便宜:身份 / 自我认知层自带传播(镜子、印象箱),不靠买量。
- 捆绑有溢价:卖「一张卡 = 一堆 AI 私教随便用 + 让我被更多人看见」,用户不逐功能比价。
- 个性化即护城河:导师读你的分身画像,按你的职业 / 生意教 —— **只有喂过分身的人用着才爽**,别人抄不走。

**两个变现口:** ① 会员订阅(主线) ② 服务者获客增值(垂直切口)。

## 现有功能怎么归位(不是推倒重来)

| 已有 / 在做 | 在新叙事里的角色 |
|---|---|
| PersonaAI 分身 | **活名片本体(核心)** |
| 问答箱 | 别人来问这张活名片 |
| escalate(AI 答不上转本人) | 名片本人补一句 |
| 镜子分身 | 名片帮你照见自己(对内) |
| 印象箱(待做) | 别人帮你补全名片(对外、双边拉新) |
| 找对象测评 | 名片在相亲场景的一个用法 |
| ToolAgent(英语 / 面试) | **会员变现器官**:升级为接画像的 AI 导师 |

## 四条设计铁律

1. 卖**会员捆绑**,绝不单卖某个工具(单卖就掉回红海)。
2. 每个 AI 导师**必接分身画像**做个性化,否则护城河归零。
3. 身份与学习统一在「**投资你自己**」一句话下,不违和。
4. 导师**数量克制、个性化深** > 数量多;它是变现器官,不是又一轮「加 Agent」。

## 头 90 天验证顺序

- **0–30 天｜身份核心立住**:活名片 + 镜子。验证免费层病毒系数(分享率 / K 因子)。
- **30–60 天｜让名片长出脚**:上印象箱(双边拉新)+ 把分发物料做顺(专属码 / 海报 / 朋友圈文案 / 微信快捷回复)。验证「人真的愿意把活名片发出去」。
- **60–90 天｜验证付费**:上 1–2 个 AI 导师(先**英语**——付费心智最熟;再**商业**——契合做生意的服务者),接画像,小范围验证免费→会员转化率。
- **决策点**:转化率达标 → 扩导师 / 扩曝光变现;不达标 → 回身份层补留存,别急着堆功能。

## 生死线指标

- **增长**:免费层分享率 / K 因子。
- **变现**:免费→会员**转化率**(这是命)。
- **留存**:分身喂养深度(越喂越懂 = 粘性)。

---
*先证明有人肯付,再谈规模。身份层再火,没人为「被看见 + 变更好」掏钱,飞轮就空转。*
